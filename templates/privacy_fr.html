Politique de Confidentialité

Dernière mise à jour : 6 juillet 2026

1. Qui Sommes-Nous

Laval Digital (« nous », « notre », « nos ») exploite {{ agent_name }}, un assistant marketing IA pour les boutiques Shopify. Nous sommes basés à Laval, Québec, Canada.

2. Quelles Données Nous Collectons

Données du compte : Adresse courriel, nom, domaine de la boutique Shopify, informations de facturation traitées sécuritairement via notre processeur de paiement.

Données Shopify : Avec votre autorisation via OAuth, nous accédons aux produits, commandes, clients, analyses, thèmes, rabais, exécutions et événements marketing de votre boutique Shopify — seulement les données nécessaires pour fournir le service marketing.

Données Google Ads : Lorsque vous ajoutez notre MCC comme partenaire, nous accédons aux données de vos campagnes Google Ads (campagnes, mots-clés, mesures de rendement) via l'API Google Ads pour gérer et optimiser votre publicité.

Données d'utilisation : Historique de conversations avec les agents IA, journaux d'exécution de tâches et préférences de campagne. Ces données sont utilisées uniquement pour améliorer vos résultats marketing.

Témoins essentiels : Nous utilisons uniquement des témoins de session essentiels pour l'authentification. Aucun témoin de suivi, d'analyse ou de marketing tiers.

3. Comment Nous Utilisons Vos Données

Nous utilisons vos données exclusivement pour : (a) opérer le service marketing IA, (b) exécuter les tâches marketing que vous demandez, (c) améliorer la qualité du service, et (d) respecter les obligations légales. Nous ne vendons jamais vos données à des tiers.

4. Partage des Données

Nous partageons les données uniquement avec :

  • Shopify — via API OAuth, uniquement pour exécuter vos tâches marketing
  • Google Ads — via l'API Google Ads, uniquement pour gérer vos campagnes
  • DeepSeek / LiteLLM — les fournisseurs de modèles IA traitent vos requêtes pour générer du contenu marketing
  • Processeur de paiement — pour la facturation (nous ne stockons pas les détails de carte de crédit)

5. Conservation des Données

Nous conservons vos données tant que votre compte est actif. Lors de l'annulation, vos données sont supprimées dans les 30 jours. Les jetons OAuth Shopify sont révoqués immédiatement après la désinstallation.

6. Vos Droits (RGPD / LPRPDE)

Vous avez le droit d'accéder, d'exporter, de corriger ou de supprimer vos données personnelles en tout temps. Utilisez l'outil d'exportation RGPD dans votre tableau de bord ou écrivez-nous à {{ CONTACT_EMAIL }}.

7. Sécurité

Toutes les données sont chiffrées en transit (TLS 1.3) et au repos. Les identifiants sont chiffrés à l'aide de Fernet (chiffrement symétrique) avec dérivation de clé PBKDF2.
